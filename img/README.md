# Imágenes — ivanro.com

Suelta tus fotos aquí con **exactamente estos nombres**. La web las toma
automáticamente; mientras no existan, se muestra un placeholder elegante.

| Archivo | Dónde aparece | Recomendación |
|---|---|---|
| `ivan-portrait.jpg` | Foto circular del hero (lo primero que se ve) | Primer plano, cara clara, cuadrada (ej. 800×800) |
| `ivan-trabajo.jpg`  | Sección 02 "Lo que cambió" | Tú trabajando remoto / con el laptop. Vertical 4:5 |
| `ivan-campo.jpg`    | Sección 03 "Quién soy hoy" | Tú en el campo / con tus perros. Vertical 4:5 |

- Formato: `.jpg` (o cambia la extensión en el HTML si usas `.png`/`.webp`).
- `ivan-portrait.jpg` también se usa como imagen de previsualización al
  compartir el link (Open Graph / Twitter card).
- Pesos sugeridos: < 300 KB cada una para que cargue rápido.
